# Paper Reading Club

Компания всех заинтересованных в чтении и анализе статей из области распределёных СУБД. Неполный список тематик:

- consistency protocols;
- transaction protocols;
- storage algorithms and data structures;
- concurrent algorithms and data structures;
- query optimisations;
- distributed computing;

Предложения можно слать лично или через PR-ы.
